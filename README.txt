# 📊 Análisis de Ventas Retail con Python

## 🚀 Descripción

Proyecto de análisis de ventas donde se procesan datos de ventas, inventario y costos para obtener métricas clave del negocio.

## 🔧 Tecnologías

* Python
* Pandas
* Matplotlib
* Power BI

## 📈 Métricas calculadas

* Ganancia total
* Productos más rentables
* Ratio de ventas vs stock
* Alertas de inventario

## 📊 Dashboard

Se desarrolló un dashboard en Power BI para visualizar:

* Rentabilidad por producto
* Ventas vs stock
* Alertas de reposición

## 🧠 Insights

* Identificación de productos con alta demanda
* Detección de sobrestock
* Optimización de inventario

## 📁 Estructura

* analisis.py → script principal
* CSV → datos
* reporte_general.csv → salida
